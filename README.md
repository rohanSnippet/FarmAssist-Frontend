# FarmAssist — Frontend Repository

![home page](https://repository-images.githubusercontent.com/1105741573/b1dbf559-1119-4cc7-b5bf-a98583009f96)

---

FarmAssist is a web-based platform designed to simplify agricultural processes and provide crop-related insights for users.  
This repository contains the **Frontend** built using **React + Vite**.

---

## 🛠 Tech Stack

| Technology       | Usage               |
| ---------------- | ------------------- |
| React            | UI Building         |
| Vite             | Fast Dev + Bundling |
| JavaScript / JSX | Core Development    |

---

## 🚀 Getting Started

Follow the steps below to run the project locally:

### 1️⃣ Navigate to the Frontend Directory

```bash
cd FarmAssist-Frontend
```

### 2️⃣ Install All Dependencies

```bash
npm i
```

### 2️⃣ Run the Frontend server

```bash
npm run dev
```
